# 2D飞机射击游戏

黑白风格的纵向卷轴射击游戏，专为移动端H5设计。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Phaser 3
- Vue Router
- Pinia

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 游戏特性

- 黑白复古街机风格
- 触摸控制（移动端）+ 键盘控制（桌面测试）
- 自动射击
- 多种敌人类型（小飞机、大炮、电网、Boss）
- 道具系统（血包、武器强化、护盾、加速）
- 升级系统
- 本地排行榜
